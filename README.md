# Recipe-Diary　- 模写コーディング

## プロジェクト概要
このプロジェクトは、[CodeJump様](https://www.codejump.com/) の模写コーディングです。  
HTML/CSSの学習を目的として、デザインカンプを元にコーディングしました。

## 使用技術
- HTML
- CSS

## インストール方法

1. リポジトリをクローン
git clone https://github.com/srgnair/Recipe-Diary.git

2. `index.html` をブラウザで開く
